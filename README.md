# Merge Sort Implementation and Test Instructions

This project provides a Python implementation of the Merge Sort algorithm along with a suite of unit tests to verify its functionality.

## Files

- `merge_sort.py`: Implementation of the Merge Sort algorithm.
- `test_merge_sort.py`: Unit tests for the `merge_sort` function utilizing Python's `unittest` framework.

## Setup Instructions

Follow these steps to ensure proper setup and execution of the test suite:

1. **Install Python**: Ensure Python is installed on your system. Verify installation by running:
   ```sh
   python --version
   ```
   If not installed, download and install it from [python.org](https://www.python.org/).

2. **Verify the 'unittest' Module**: The `unittest` module is included with Python's standard library. Ensure Python is installed correctly to include this module.

3. **Navigate to Project Directory**: Use your command-line interface (CLI) to change to the project directory:
   ```sh
   cd path/to/your/directory
   ```

4. **Run Tests**: Execute the test suite with the following command:
   ```sh
   python -m unittest test_merge_sort.py
   ```

## Troubleshooting

- **Correct Directory**: Confirm that you are in the directory containing both `merge_sort.py` and `test_merge_sort.py`.
- **Ensure File Integrity**: Ensure that the files `merge_sort.py` and `test_merge_sort.py` are not altered unexpectedly and are saved on your file system.
- **Python Environment Setup**: If tests fail, verify your Python setup by checking the Python path and version, and ensure that the `unittest` library is functioning.
- **Error Details**: For persistent errors, document the error message and check for typographical errors in the script or directory path for further assistance.
- **Re-run the Build**: If a build failure message occurs, consider attempting the setup again or check for issues in the build system configuration.

For additional guidance, please refer to the comments within `test_merge_sort.py` which explain specific test cases.
